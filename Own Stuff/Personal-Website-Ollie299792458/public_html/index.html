Why the name?

When I was much younger, I loved a LEGO.com game called CrystAlien Conflict. It is a flash based single player RTS, you can find an archived version here. "299792458" was a cheat code in the game, it made everything build faster. When choosing usernames I would, in my naivety, try "ollie", then discover it was taken. Not wanting to have a different number after "ollie" on each site, I went for "ollie299792458", and it has served me well ever since.

Also, it is the exact speed of light in m/s (but I only discovered this much later).
